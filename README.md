## 🎯 Projeto em Destaque

### 🐶 **[Peter Palm](https://github.com/rochadelon/peter-palm)** - O seu copiloto de desktop inteligente

  <img src="https://img.shields.io/badge/Status-Ativo%20em%20Desenvolvimento-brightgreen?style=for-the-badge" />

**Peter Palm** é uma aplicação desktop que funciona como extensão digital da sua mente, offering assistência proativa com foco total em privacidade local-first.

#### ✨ Principais características:

| Recurso | Descrição |
|---------|----------|
| 🤖 **Inteligência Contextual** | Integração com OpenRecall para memória digital privada e local |
| ⏱️ **Contexto Ajustável** | Escolha 2min, 1h, 1 dia ou histórico infinito para análise |
| 🧠 **Personalidade Adaptável** | Modo Listen herda sua personalidade customizável ("Palms") |
| 🇧🇷 **100% Português** | Interface e insights completamente localizados para PT-BR |
| 🏢 **Assistente de Reuniões** | Transcrição em tempo real, resumos automáticos e extração de tarefas |
| 🔒 **Processamento Local** | Sem dados trafegando para nuvem, você controla suas chaves |

#### 🔧 Stack tecnológico:
- **Frontend**: Electron + LitElement (Web Components)
- **Backend**: FastAPI (Python) + PyTorch + Sentence-Transformers
- **Memória**: OpenRecall + ChromaDB + SQLite
- **Modelos**: OpenAI, Google Gemini, Claude + Ollama (local)
- **IA**: Whisper, Doctr (OCR), Deepgram (STT)

🔗 **Conexões**: Veja tecnologias similares em [Stack Tecnológico](#-stack--ferramentas) e projetos relacionados em [Machine Learning & Visão Computacional](#-machine-learning--visão-computacional)
